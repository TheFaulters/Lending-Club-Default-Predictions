Sentiment Analysis¶

Aguillar, Goswami, Fallon

In this notebook we perform sentiment analysis to assess to what extent the descriptions provided in each loan allow us to predict risky loans. We use a simple Logistic Regression model to test whether this method yields any promising results. While we do not find that descriptions are useful predictors, we believe the use of more advanced natural language processing techniques may yield better results.

The structure of this notebook is as follows:

  1. Importing Data and Packages

  2. Cleaning

  3. Visual Analysis of Cleaned Data

  4. Model Optimization

  5. Final Results
